📘 Signalist — Real‑Time Stock Market Insights & AI‑Powered Watchlists
Signalist is a modern, high‑performance stock market analytics platform built with Next.js 14, BetterAuth, MongoDB, Finnhub, Gemini AI, and Inngest.
It provides real‑time stock data, personalized watchlists, AI‑generated insights, and automated background jobs — all deployed on Vercel.

🚀 Features
🔐 Authentication
Google OAuth via BetterAuth

Secure session handling

User data stored in MongoDB

📈 Real‑Time Stock Data
Live quotes from Finnhub API

Key metrics, charts, and market info

Dynamic stock detail pages (/stocks/[symbol])

⭐ Personalized Watchlist
Add/remove stocks with one click

Watchlist stored per user

Fully synced across devices

🤖 AI‑Powered Insights
Daily summaries generated by Google Gemini

Trend detection & sentiment analysis

Smart recommendations

🕒 Background Jobs
Automated tasks using Inngest

Daily email summaries

Scheduled data processing

📬 Email System
Transactional emails via Resend

Secure login links

HTML templates
